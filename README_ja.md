# z - jump around

このプロジェクトは **`z`** と呼ばれる、コマンドラインでのディレクトリ移動を効率化するためのツールです。

## 主な機能と特徴

*   **「Frecency」に基づく移動:**
    *   「Frecency」とは、**Frequency（頻度）**と **Recency（直近のアクセス）**を組み合わせた造語です。
    *   あなたがよく訪れるディレクトリや、最近訪れたディレクトリを学習し、そのランクに基づいて移動先を決定します。
*   **高速なディレクトリジャンプ:**
    *   長いパスをすべて入力しなくても、ディレクトリ名の一部（正規表現）を入力するだけで、最も可能性の高いディレクトリへ瞬時に `cd`（移動）できます。
    *   例: `/home/user/work/project/src` に移動したい場合、過去に何度か訪れていれば `z src` や `z pro src` などの短いコマンドで移動可能です。
*   **学習機能:**
    *   通常の `cd` コマンドでの移動をバックグラウンドで追跡し、データベース（デフォルトでは `~/.z`）に蓄積します。使えば使うほど精度が向上します。
*   **柔軟なオプション:**
    *   `-r`: ランク（頻度）のみを重視して移動。
    *   `-t`: 直近のアクセスのみを重視して移動。
    *   `-l`: 移動せずに、マッチするディレクトリのリストを表示。
    *   `-c`: カレントディレクトリのサブディレクトリに限定して検索。
*   **互換性:**
    *   `bash` および `zsh` で動作します。

## 使い方（概要）

1.  `.bashrc` や `.zshrc` に `z.sh` を読み込む設定を追加します（例: `. /path/to/z.sh`）。
    *   内部では Python で実装された `alt-z` コマンドが呼び出されます。
2.  しばらく普通に `cd` コマンドを使ってディレクトリ移動を行い、データベースを構築します。
3.  `z <キーワード>` で目的のディレクトリへジャンプできるようになります。

簡単に言えば、**「よく使うディレクトリを覚えてくれて、短いコマンドで飛べるようにしてくれる賢い `cd` コマンド」**です。
